# NaidiW

## Inicializar el repositorio

```bash
Inicio > Windows > Terminal
cd C:\
cd naidiw
mkdir NaidiW
dotnet new sln -n NaidiW
dotnet new webapi -o NaidiW.API
dotnet sln add NaidiW.API
dotnet build
dotnet new classlib -n NaidiW.Business
dotnet sln add NaidiW.Business
```
